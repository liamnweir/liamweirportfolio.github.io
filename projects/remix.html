Portfolio Writeup (Professional)

Use this verbatim on your project page
ProblemMIDI ideas need fast iteration

Creators often start with a melody but need harmonies, bass motion, and variations quickly—without a full DAW loop. This project converts a raw MIDI idea into arranged alternatives in seconds.

What it doesRemix pipeline
  • Analyze: melody-like track selection, tempo + key estimate, bar segmentation.
  • Generate: bar-wise chord fitting & voicing, bassline, counter-melody, rhythm transforms.
  • Evaluate: embedded randomized A/B tests + exportable dataset + plots.
  • Ship: plays in browser + downloads as MIDI.
ArchitectureBrowser-only
UI
 ├─ MIDI parse/write (tonejs/midi)
 ├─ Remix engine (rules + style presets)
 ├─ Playback (Tone.js)
 └─ Study + evidence (localStorage + plots)
Suggested deliverablesHiring magnet
  • 30–60s demo video (upload + remix + A/B + download)
  • 3–5 before/after MIDI examples (and optional rendered audio)
  • Results screenshot: evidence plots + interpretation paragraph
  • Repo instructions: run locally + reproduce plots
